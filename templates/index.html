About

화재란?

'불에 의한 재난'으로 소화설비를 이용하여 소화할 필요가 있는 연소현상, 화학적 폭발을 의미


현대사회에서 소방안전장치의 필요성 대두


이 프로젝트는 위험 상황으로부터 우리를 구해줄 안전장치의 존재여부를 파악하기 위함으로
건물 내부를 포함한 주변 공간에 비치된 소방안전장치의 위치를 확인하도록 권고

Model

Yolo


"You Only Look Once"

-> Object Detection에서 많이 알려진 모델

1. 이미지 전체를 한번만 봄(이미지 여러장 분할 X)

2. 통합된 모델을 사용해 간단함

3. 실시간으로 객체 탐지 가능

VS

Mask R-CNN


Faster R-CNN + mask branch
-> Instance Segmentation

1. 각 RoI의 feature extraction에 RoIPool 대신 RoIAlign을 사용

2. mask prediction과 class prediction을 분리(decoupling)

3. Faster R-CNN에 간단하게 작은 FCN을 추가해주어 다양한 computer vision tasks를 가능하게 함
